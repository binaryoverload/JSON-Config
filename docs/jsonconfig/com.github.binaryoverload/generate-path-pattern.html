generatePathPattern

fun generatePathPattern(pathSeparator: Char, allowedSpecialCharacters: Set<Char>): Pattern

Generate a Pattern (compiled) from pathSeparator char and allowedSpecialCharacters char array. You should only run this when it is needed and store the return. You should NOT run this multiple times!

Parameters

pathSeparator - The path separator char to include in the regex.

allowedSpecialCharacters - The allowed special characters which will be escaped. Keep in mind this will not verify a conflict. See .verifyNoConflict for that.

Return
The compiled Pattern used for checking a valid path with the specified pathSeperator and allowedSpecialCharacters.
