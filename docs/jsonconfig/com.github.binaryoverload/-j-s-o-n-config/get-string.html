@JvmOverloads fun getString(path: String, forceConversion: Boolean = false): String?

Gets a string at the specified path

Parameters

path - The path to get the string at.

forceConversion - Whether to forcefully convert the element to a string or not. If this is false, this method will return IllegalStateException for any element that isn't a JSON String.

Exceptions

IllegalArgumentException - if the path is malformed.

IllegalStateException - if the element at the path is not a string.

Return
The string value. If the value at the path does not exist then this returns null.
