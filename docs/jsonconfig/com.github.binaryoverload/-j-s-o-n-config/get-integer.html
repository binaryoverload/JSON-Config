fun getInteger(path: String): Int?

Gets a integer at the specified path

Parameters

path - The path to get the integer at.

Exceptions

IllegalArgumentException - if the path is malformed.

IllegalStateException - if the element at the path is not a number.

Return
The integer value at this path. If the value at the path does not exist then null is returned.
