getElement

fun getElement(json: JsonObject, path: String): JsonElement?

Method to get a JSON Element from a specified path

It is not recommended to use this method! Usecom.github.binaryoverload.JSONConfig.getElement instead!

Parameters

json - The object to search in

path - The path to get the element from. If this is blank, it returns the entire object. If the path is malformed, then it throws an IllegalArgumentException

Exceptions

IllegalArgumentException - if the path is malformed

Return
The element at the specified path. Returns null if the element doesn't exist.
