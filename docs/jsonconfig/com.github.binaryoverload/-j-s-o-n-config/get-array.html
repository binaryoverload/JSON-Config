fun getArray(path: String): JsonArray?

Gets a JSON array at the specified path

Parameters

path - The path to get the array at.

Exceptions

IllegalArgumentException - if the path is malformed

IllegalStateException - if the element at the path is not an array

Return
The array at this path. If the value at the path doesn't exist then null is returned.
