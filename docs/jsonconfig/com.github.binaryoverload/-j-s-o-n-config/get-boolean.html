fun getBoolean(path: String): Boolean?

Gets a boolean at the specified path

Parameters

path - The path to get the boolean at.

Exceptions

IllegalArgumentException - if the path is malformed.

IllegalStateException - if the element at the path is not a boolean.

Return
The boolean value at this path. If the value at the path does not exist then null is returned.
